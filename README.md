# Maitri's Portfolio Website

A personal portfolio website showcasing my journey as a Quality Assurance Engineer. This project represents my professional experience, skills, projects, and achievements in the software testing industry.

## What I Built

This is a fully responsive, single-page portfolio website that tells my story through:
- **Hero Section**: Introduction and key stats
- **My Story**: Personal background and work philosophy
- **What I Bring**: Skills and services I offer
- **Professional Journey**: Timeline of work experience
- **Key Achievements**: Featured projects and professional milestones
- **Certifications**: Interactive carousel of certifications
- **Contact**: Form with email integration

## Technologies Used

I kept it simple and focused on the fundamentals:

- **HTML5**: Semantic markup for accessibility and SEO
- **CSS3**: Custom styling with CSS variables for theming
- **JavaScript**: Vanilla JS for interactivity (no frameworks)
- **Formspree**: For contact form email functionality

## How I Created This Website

I started with a basic HTML structure and built it up section by section. The process went something like this:

1. **Planning**: I mapped out what sections I wanted and what content each should have
2. **Structure**: Created the HTML skeleton with semantic tags
3. **Styling**: Built the CSS from scratch, focusing on a dark theme with blue accents
4. **Content**: Wrote personal stories, project descriptions, and achievement details
5. **Interactivity**: Added JavaScript for navigation, carousel, and form handling
6. **Images**: Organized certification and achievement photos in an images folder
7. **Refinement**: Tweaked spacing, fonts, and colors for a polished look

## Key Features

- **Responsive Design**: Works on mobile, tablet, and desktop
- **Dark Theme**: Easy on the eyes with a modern aesthetic
- **Interactive Carousel**: Swipeable certifications section
- **Contact Form**: Sends emails directly to my inbox via Formspree
- **Smooth Navigation**: Single-page scrolling with active section highlighting
- **GitHub Integration**: Project hosted on GitHub with proper version control

## Getting Started

To view or modify this website:

1. Clone the repository
2. Open `index.html` in your browser
3. Edit any of the files to customize content

## Customization Tips

If you want to use this as a template for your own portfolio:

- Replace personal information in the contact section
- Update the Formspree form endpoint with your own
- Swap out images with your own projects and certifications
- Adjust colors in the CSS variables at the top of styles.css
- Modify the content to tell your own story

## Deployment

This site is hosted on GitHub Pages. To deploy your own version:

1. Create a GitHub repository
2. Push your files
3. Enable GitHub Pages in repository settings
4. Choose the main branch as source
