# Portfolio Maitri

A lightweight portfolio starter built with HTML, CSS, and JavaScript.

## Getting started

- Open `index.html` in your browser
- Edit the content in `index.html`, `styles.css`, and `script.js`

## Structure

- `index.html` — main page markup
- `styles.css` — responsive styling
- `script.js` — simple smooth scroll behavior

## Next steps

- Add your projects in the `#projects` section
- Replace the contact email in the `#contact` section
- Add a `profile.jpg` or other media files as needed
